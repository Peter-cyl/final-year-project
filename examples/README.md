# Example Domain: Refrigeration

This folder contains the reference refrigeration domain used for development and testing.

## Files:

- `refrigerated_delivery_domain.pddl` - The planning domain definition
- `refrigerated_delivery_problem.pddl` - A sample problem instance
- `plan.pddl` - An example solution plan

## Reference:

This is the primary test case for the PlanVerb project. Your verbalizer should be able to:
- Parse this domain
- Extract predicate and action comments
- Generate natural language explanations for queries like `predicate-refrigeratedttruck`
