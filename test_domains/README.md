# Test Domains

These PDDL planning domains are provided for testing the PlanVerb NLG system.

## Available Domains:

- **Refrigeration** - Multiple variants of the refrigerated delivery domain
- **Rovers** - Mars rover planning domain
- **Satellite** - Satellite observation planning
- **Settlers** - Colony building domain
- **CrewPlanning** - Crew scheduling
- **Original/Constrained** - Different constraint levels

## Usage:

Use these domains to test your verbalizer's ability to:
1. Parse different PDDL structures
2. Handle various predicate types
3. Generate explanations for different planning scenarios
